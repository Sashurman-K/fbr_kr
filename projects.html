Интернет-магазин

HTML/CSS JavaScript PHP MySQL

Описание проекта

Полнофункциональный интернет-магазин электроники с системой управления товарами, корзиной покупок и интеграцией с платежными системами.

Проект включает административную панель для управления товарами, заказами и пользователями. Реализована система фильтрации и поиска товаров, отзывы и рейтинги.

Основные функции

  • Каталог товаров с фильтрацией
  • Корзина покупок
  • Система оплаты
  • Личный кабинет пользователя
  • Административная панель

Корпоративный сайт

React Bootstrap API Node.js

Описание проекта

Современный адаптивный корпоративный сайт для IT-компании с анимациями, блогом и системой управления контентом.

Сайт включает разделы "О компании", "Услуги", "Портфолио", "Блог" и "Контакты". Реализована админ-панель для управления контентом.

Основные функции

  • Адаптивный дизайн
  • Система управления контентом
  • Блог с комментариями
  • Форма обратной связи
  • Интеграция с социальными сетями
